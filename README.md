# Dragons-Hoard-Inter-Reality-Banking
A fun Python project I am working on for DM's and players to track their gold and In-game spending in a more immersive way!
95% of code is handwritten, not AI! 
  #  Launch feature goals:
    #  saved data Locally
    #  gold converter
    #  formatted recipt .txt file writer
    # sort or search purchases
    # accounts for DM's with party gold total/spending logged, DM Loans ( d20 table on what intrest (one chicken leg, or 5X the loan back) is, as it is run by goblins)
    # logs Deposits and withdrawls on DM side, and updates party gold supply

  # Potiental Future Ideas
    cloud saves and real time acounts that the DM and players can update and manage with different settings by roles
    personalization and settings
    established lore and locations to visit and use in game!
    having the app react to your charater and remember features about them, as Dragon's Hoard banking cares for each member!(Dm can log or adapt accounts or add quirks to the player interface and services)
  # Use-case and Ideas to incorperate in play
    This program is meant to be a fun and flexible tool, so it can be as included and immersive as players and the DM would like. Use it simply as a tracker and converter. or a real "accurate" banking service the charaters must navigate. (or take ownership of, Rob, work for, save from Robbers, run side quests from, visit and explore, etc no judgement) 
    
    As we all know, DND Beyond and other table top tools track inventory and gold, but what if this app actually worked as a bank? Any gold tracked through those platforms could be considered "loose change" physically on the charater and included in weight calculations if that is tracked in your campign, subject to be lost or stolen, or some transactions must be made without credit. The bank can make transactions on the charaters behalf,if the service is in your game world, maybe allow them to have an established presence in large towns and cities and the store can make inqury on the account and talk to Dragons Hoard directly to be reimbursed.
